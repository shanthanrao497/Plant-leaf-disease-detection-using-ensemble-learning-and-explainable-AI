# Plant-leaf-disease-detection-using-ensemble-learning-and-explainable-AI

Description:
AI-powered web application that identifies plant leaf diseases from uploaded images using Ensemble Learning models. The system improves prediction accuracy by combining multiple machine learning algorithms and provides transparent predictions through Explainable AI (XAI) techniques.

Technologies:
Python, Flask, Machine Learning, Ensemble Learning, Explainable AI (XAI), HTML, CSS, JavaScript, NumPy, Pandas, Scikit-learn, OpenCV

Features:

* Plant disease detection from leaf images
* Image upload through web interface
* Ensemble model-based classification
* Explainable AI visualizations
* Real-time disease prediction
* User-friendly dashboard
* Fast and accurate diagnosis
* Disease information display
* Responsive web application

System Workflow:

1. User uploads a leaf image.
2. Image preprocessing is performed.
3. Features are extracted from the image.
4. Ensemble learning models analyze the image.
5. Disease prediction is generated.
6. XAI module explains the prediction.
7. Results are displayed on the web interface.

Benefits:

* Early disease detection for crops
* Improved prediction accuracy
* Transparent AI decision-making
* Easy to use for farmers and researchers
* Supports better agricultural productivity

How to Run:

1. Clone the repository.
2. Install required dependencies using:
    pip install -r requirements.txt
3. Run the application:
    python app.py
4. Open the browser and visit:
    http://localhost:5000
